Tinker Board 3N Android14 WIFI固定IP设定

2025-09-09 Android, Tinker 0 评论

  • 固件版本：

Tinker Board 3N Android 14 V1.0.15

  • 操作步骤：

1、正常连接需要的WIFI SSID，比如测试WIFI：OPENCLEB，进入当前已连接WIFI设定

2、进入 网络详情，右上角 修改 页面

3、进入当前WIFI高级选项

4、IP设置，默认：DHCP，点击修为：静态

5、根据路由器的分配原则，手动添加IP地址、网关、网络前缀长度（子网掩码换算器）、DNS等

6、如上信息如果不知道路由器的规则，可在DHCP模式下，记录自动分配地址等信息后，参照填入
