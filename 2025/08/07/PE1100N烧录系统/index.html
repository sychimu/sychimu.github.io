PE1100N烧录系统

2025-08-07 PE1100N Jetson 评论

PE1100N仅支持通过 Ubuntu 烧录镜像，不支持通过 Windows 烧录镜像。

1.1 恢复模式

  1. 系统要求

    • Linux 主机（x86 Ubuntu 18.04 及更高版本）

    • Micro USB 数据线

  2. 进入强制恢复模式
    PE1100N Flash端口是数字 ❷，强制恢复按钮是数字 ❹

请执行以下步骤PE1100N进入强制恢复模式：

[PE1100N]

  1. 关闭PE1100N电源并拔下电源线。
  2. 使用 Micro USB 数据线连接主机和PE1100N Flash端口（数字 ❷）。
  3. 用细针顶住强制恢复按钮（数字 ❹）。
  4. 连接电源线并打开PE1100N电源。
  5. 3 秒后松开强制恢复按钮。

[主机]

在 Linux 主机 PC 上，打开终端窗口并输入命令 。如果返回的内容根据您使用的 Jetson SoM 具有以下输出之一，则该板处于强制恢复模式。

lsusb

• Orin NX 16GB：0955：7323 NVidia Corp
• Orin NX 8GB：0955：7423 NVidia Corp
• Orin Nano 8GB：0955：7523 NVidia Corp
• Orin Nano 4GB：0955：7623 NVidia Corp

1.2 烧录镜像

[主机]

  1. 在主机上提取 BSP 文件

    BSP 文件示例：

系统 文件名
PE1100N Orin Nano 4GB PE1100N Orin Nano 4GB JetPack x.x.x Image Vx.x.x
PE1100N Orin Nano 8GB PE1100N Orin Nano 8GB JetPack x.x.x Image Vx.x.x
PE1100N Orin NX 8GB PE1100N Orin NX 8GB JetPack x.x.x Image Vx.x.x
PE1100N Orin NX 16GB PE1100N Orin NX 16GB JetPack x.x.x Image Vx.x.x

下载链接 ： https://www.asus.com/networking-iot-servers/aiot-industrial-solutions/embedded-computers-edge-ai-systems/pe1100n/helpdesk_download?model2Name=PE1100N

  1. 更改文件夹，执行烧录命令
  1. 烧录镜像大约需要 15 分钟

[PE1100N]
15 分钟后，PE1100N将自动重启。

注意：

  1. 请勿在主机和PE1100N之间使用 USB 集线器。
  2. 您可以从“mfi_PE1100N-orin/initrdlog”中了解刷新图像的过程。
